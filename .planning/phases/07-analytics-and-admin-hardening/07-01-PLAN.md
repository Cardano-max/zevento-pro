Create `api/src/admin/dto/analytics-query.dto.ts`:
```typescript
import { IsDateString, IsOptional } from 'class-validator';

export class AnalyticsQueryDto {
  @IsOptional()
  @IsDateString()
  dateFrom?: string;

  @IsOptional()
  @IsDateString()
  dateTo?: string;
}
```

Add `getAnalyticsDashboard(query: AnalyticsQueryDto)` to AdminService.

The method must:
1. Compute `since` = `query.dateFrom ? new Date(query.dateFrom) : new Date(Date.now() - 24 * 60 * 60 * 1000)` and `until` = `query.dateTo ? new Date(query.dateTo) : new Date()`.
2. Run four Prisma calls in `Promise.all`:
   - `this.prisma.lead.groupBy({ by: ['city'], _count: { id: true }, where: { createdAt: { gte: since, lte: until } }, orderBy: { _count: { id: 'desc' } } })` → leadsPerCity
   - `this.prisma.lead.groupBy({ by: ['status'], _count: { id: true }, where: { createdAt: { gte: since, lte: until } } })` → funnelRaw
   - `this.prisma.transaction.groupBy({ by: ['type'], _sum: { amountPaise: true, commissionPaise: true }, _count: { id: true }, where: { status: 'PAID', createdAt: { gte: since, lte: until } } })` → revenueRaw
   - `this.prisma.vendorProfile.count({ where: { status: 'APPROVED', subscription: { status: { in: ['ACTIVE', 'AUTHENTICATED'] } } } })` → activeVendorCount
3. Normalize funnelRaw into an ordered array. Define constant `FUNNEL_ORDER = ['PENDING', 'ROUTING', 'ROUTED', 'NOTIFIED', 'QUOTES_RECEIVED', 'BOOKED', 'COMPLETED', 'CANCELLED']`. Map over FUNNEL_ORDER, find matching funnelRaw entry (default count 0 if absent):
   ```typescript
   const funnelMap = new Map(funnelRaw.map((r) => [r.status, r._count.id]));
   const conversionFunnel = FUNNEL_ORDER.map((status) => ({ status, count: funnelMap.get(status) ?? 0 }));
   ```
4. Normalize revenueRaw into revenueByStream array: `revenueRaw.map((r) => ({ type: r.type, count: r._count.id, totalAmountPaise: r._sum.amountPaise ?? 0 }))`.
5. Return: `{ window: { from: since, to: until }, leadsPerCity: leadsPerCity.map((r) => ({ city: r.city, count: r._count.id })), conversionFunnel, revenueByStream, activeVendorCount }`.

Add import for `AnalyticsQueryDto` at top of admin.service.ts.
