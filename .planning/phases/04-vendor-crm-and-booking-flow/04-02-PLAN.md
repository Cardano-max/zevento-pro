    **File: api/src/quote/quote.service.ts**
    Injectable service with PrismaService and `@InjectQueue('quote-expiry') private quoteExpiryQueue: Queue`.

    Method `createOrUpdateQuote(leadId: string, vendorId: string, dto: CreateQuoteDto)`:
    - Check if quote already exists for (leadId, vendorId) in DRAFT status
    - If exists in DRAFT: delete existing line items, update quote fields
    - If SUBMITTED/ACCEPTED/etc: throw BadRequestException('Quote already submitted — cannot edit')
    - If no quote exists: create Quote with status DRAFT
    - Calculate totalPaise = sum of (lineItem.amountPaise * lineItem.quantity) from dto.lineItems
    - Create/replace QuoteLineItem records
    - Return quote with lineItems included
    - Use upsert logic: `prisma.quote.upsert({ where: { leadId_vendorId: { leadId, vendorId } }, ... })` — Prisma generates this compound unique accessor from @@unique([leadId, vendorId])

    Method `submitQuote(quoteId: string, vendorId: string)`:
    - Uses `prisma.$transaction(async (tx) => {...})`
    - `tx.quote.updateMany({ where: { id: quoteId, vendorId, status: 'DRAFT' }, data: { status: 'SUBMITTED', submittedAt: new Date() } })` — if count === 0 throw BadRequestException
    - Fetch quote to get leadId and validUntil
    - Check if this is the FIRST submitted quote for this lead:
      `const submittedCount = await tx.quote.count({ where: { leadId, status: 'SUBMITTED', id: { not: quoteId } } })`
      If submittedCount === 0: `tx.lead.update({ where: { id: leadId }, data: { status: 'QUOTES_RECEIVED' } })`
    - Return updated quote
    - OUTSIDE transaction: enqueue BullMQ delayed expiry job:
      ```typescript
      const delayMs = Math.max(0, quote.validUntil.getTime() - Date.now());
      await this.quoteExpiryQueue.add('expire-quote', { quoteId: quote.id }, {
        delay: delayMs, attempts: 3, backoff: { type: 'exponential', delay: 1000 }
      });
      ```

    Method `acceptQuote(quoteId: string, customerId: string)`:
    - Uses `prisma.$transaction(async (tx) => {...})`
    - `tx.quote.updateMany({ where: { id: quoteId, status: 'SUBMITTED' }, data: { status: 'ACCEPTED' } })` — if count === 0 throw BadRequestException
    - Fetch accepted quote: `tx.quote.findUnique({ where: { id: quoteId }, include: { lead: true } })`
    - Verify `quote.lead.customerId === customerId` — throw ForbiddenException if not their lead
    - Reject other quotes: `tx.quote.updateMany({ where: { leadId: quote.leadId, id: { not: quoteId }, status: 'SUBMITTED' }, data: { status: 'REJECTED' } })`
    - Create Booking:
      ```typescript
      const booking = await tx.booking.create({
        data: { leadId: quote.leadId, quoteId, customerId, vendorId: quote.vendorId, status: 'BOOKED' }
      });
      ```
    - Create initial BookingStatusHistory: `tx.bookingStatusHistory.create({ data: { bookingId: booking.id, fromStatus: null, toStatus: 'BOOKED' } })`
    - Update Lead.status to BOOKED: `tx.lead.update({ where: { id: quote.leadId }, data: { status: 'BOOKED' } })`
    - Return booking
    - OUTSIDE transaction: increment VendorStats.totalLeadsWon:
      `prisma.vendorStats.update({ where: { vendorId: quote.vendorId }, data: { totalLeadsWon: { increment: 1 } } })`

    Method `getQuotesForLead(leadId: string, customerId: string)`:
    - Verify the lead belongs to customerId (throw ForbiddenException otherwise)
    - Return SUBMITTED quotes with line items and vendor business name:
      `prisma.quote.findMany({ where: { leadId, status: 'SUBMITTED' }, include: { lineItems: true, vendor: { select: { businessName: true, averageRating: false } } }, orderBy: { totalPaise: 'asc' } })`
    - Return array of quotes for customer comparison

    **File: api/src/quote/quote.processor.ts**
    BullMQ WorkerHost for the `quote-expiry` queue:
    ```typescript
    import { Processor, WorkerHost } from '@nestjs/bullmq';
    import { Job } from 'bullmq';
    import { Injectable, Logger } from '@nestjs/common';
    import { PrismaService } from '../prisma/prisma.service';

    @Processor('quote-expiry')
    @Injectable()
    export class QuoteExpiryProcessor extends WorkerHost {
      private readonly logger = new Logger(QuoteExpiryProcessor.name);

      constructor(private readonly prisma: PrismaService) { super(); }

      async process(job: Job<{ quoteId: string }>): Promise<void> {
        const { quoteId } = job.data;
        // IDEMPOTENT: only expires if still SUBMITTED — if already ACCEPTED, count=0, no-op
        const result = await this.prisma.quote.updateMany({
          where: { id: quoteId, status: 'SUBMITTED' },
          data: { status: 'EXPIRED' },
        });
        if (result.count > 0) {
          this.logger.log(`Quote ${quoteId} expired`);
        } else {
          this.logger.debug(`Quote ${quoteId} expiry job fired but quote was not in SUBMITTED state — no-op`);
        }
      }
    }
    ```
