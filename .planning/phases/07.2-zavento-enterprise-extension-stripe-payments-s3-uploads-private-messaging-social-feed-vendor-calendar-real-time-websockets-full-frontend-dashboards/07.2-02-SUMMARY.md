## Feature Details

### /feed Page
- Tab switcher: All | Requirements | Vendor Offers | Showcase | General
- City filter with clear button
- Post cards: author avatar (color-hashed initials), role badge (Customer blue / Vendor rose), body text, category pill, city/date/budget badges, comment count
- Expandable comment thread per post (lazy loads GET /feed/:id/comments on first expand)
- Inline comment form (requires auth, redirects to /login if not)
- "Share to Community" button opens inline create form
- Create post form: body, category select, city, event date, budget in rupees (converted to paise on submit)
- Pagination with "Load More" button
- Shimmer skeletons during load, empty state with CTA

### /favorites Page
- Auth-gated: redirects to /login?redirect=/favorites if not logged in
- Grid of saved vendor cards (same pattern as /vendors)
- Each card: filled rose heart button (top-right) to remove, spinner during DELETE
- Optimistic removal on success
- Empty state with illustration and "Browse Vendors" CTA

### /inbox Page
- Auth-gated: redirects to /login?redirect=/inbox if not logged in
- Lists all customer inquiries from GET /leads/inquiries
- Each card: status badge, category, city, date, budget, description
- Expandable vendor assignments panel per inquiry (toggle)
- Each assignment shows vendor name, city, category, assignment status badge, "View" link to vendor profile
- Pagination support

### Vendor Detail Favorites Toggle
- On mount: calls GET /customer/favorites/:id/check to initialize filled/outline heart state
- On click (logged out): redirects to /login?redirect=/vendors/:id
- On click (logged in): toggles POST/DELETE /customer/favorites/:id with loading spinner

### Navbar Updates
- Desktop nav: Community (Users icon) always visible, Saved (Heart icon) only if isLoggedIn
- Mobile nav: same conditional Saved link rendered below other nav links

### Home Page Feed Preview
- New section between Categories and AI Planner promo
- Fetches GET /feed?limit=3 alongside vendors and categories in single Promise.all
- 3-column grid of compact post cards: author avatar, name, date, body (line-clamp-3), category badge, city, comment count
- Silent error swallow — section hidden if API fails
- "See All Posts" link to /feed
