## API Calls Map
- Dashboard: GET /vendor/profile/me + GET /vendor/earnings
- Calendar: GET /vendor/calendar?year=&month=, POST /vendor/calendar/block, DELETE /vendor/calendar/block
- Inbox: GET /vendor/inbox?page=&limit=, PATCH /vendor/inbox/:id/accept, PATCH /vendor/inbox/:id/decline
- Bookings: GET /vendor/earnings (recentBookings), PATCH /booking/:id/status
- Reviews: GET /vendor/reviews
- Profile: GET /vendor/profile/me, PATCH /vendor/profile/me, POST /vendor/profile/photos
