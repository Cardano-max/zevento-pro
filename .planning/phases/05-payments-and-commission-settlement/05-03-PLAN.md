1. `getPaymentLog(page, limit, filters: { dateFrom?, dateTo?, vendorId?, type? })`:
   - Build Prisma `where` clause:
     - `type`: if provided, filter by TransactionType (validate against enum)
     - `vendorId`: join via booking.vendorId OR vendorSubscription.vendorId. Simplest: use a raw-ish approach -- query transactions and include relations, then filter. Or: use conditional where:
       ```typescript
       const where: any = {};
       if (filters.type) where.type = filters.type;
       if (filters.dateFrom || filters.dateTo) {
         where.createdAt = {};
         if (filters.dateFrom) where.createdAt.gte = new Date(filters.dateFrom);
         if (filters.dateTo) where.createdAt.lte = new Date(filters.dateTo);
       }
       if (filters.vendorId) {
         where.OR = [
           { booking: { vendorId: filters.vendorId } },
           { vendorSubscription: { vendorId: filters.vendorId } },
         ];
       }
       ```
   - `prisma.transaction.findMany({ where, orderBy: { createdAt: 'desc' }, skip, take, include: { booking: { select: { id: true, vendor: { select: { id: true, businessName: true } }, customer: { select: { id: true, name: true, phone: true } } } }, vendorSubscription: { select: { vendor: { select: { id: true, businessName: true } } } } } })`
   - Also return total count for pagination

2. `initiateRefund(dto: InitiateRefundDto)`:
   1. Find Transaction by id, verify it exists, type is BOOKING_COMMISSION, status is PAID
   2. If `dto.amountPaise` provided, validate <= transaction.amountPaise
   3. Call `razorpayService.createRefund(transaction.razorpayPaymentId, { amount: dto.amountPaise || transaction.amountPaise, notes: { reason: dto.reason, initiatedBy: 'admin' } })`
   4. Update Transaction: `status = 'REFUNDED'`
   5. Update Booking: `paymentStatus = 'REFUNDED'`
   6. Return the refund result

3. `getReconciliation()`:
   - Aggregate queries:
     ```typescript
     const totals = await this.prisma.transaction.groupBy({
       by: ['type'],
       _sum: { amountPaise: true, commissionPaise: true, netPayoutPaise: true },

       where: { status: 'PAID' },
     });
     ```
   - Payout summary: count transactions by payoutStatus (PENDING, PROCESSING, PROCESSED, FAILED, PENDING_BANK_DETAILS)
     ```typescript
     const payoutBreakdown = await this.prisma.transaction.groupBy({
       by: ['payoutStatus'],
       _sum: { netPayoutPaise: true },
       _count: true,
       where: { type: 'BOOKING_COMMISSION', status: 'PAID' },
     });
     ```
   - Return structured object:
     ```typescript
     {
       revenueByStream: totals.map(t => ({
         type: t.type,
         count: t._count,
         totalAmountPaise: t._sum.amountPaise,
         totalCommissionPaise: t._sum.commissionPaise,
         totalNetPayoutPaise: t._sum.netPayoutPaise,
       })),
       payoutBreakdown: payoutBreakdown.map(p => ({
         status: p.payoutStatus,
         count: p._count,

export class CreateCommissionRateDto {
  @IsOptional()
  @IsUUID()
  categoryId?: string;

  @IsOptional()
  @IsIn(['PLANNER', 'SUPPLIER'])
  vendorRole?: string;

  @IsInt()
  @Min(100)   // minimum 1% (100 bps)
  @Max(3000)  // maximum 30% (3000 bps)
  rateBps: number;

  @IsOptional()
  @IsDateString()
  effectiveFrom?: string;

  @IsOptional()
  @IsDateString()
  effectiveTo?: string;
}

export class UpdateCommissionRateDto {
  @IsOptional()
  @IsInt()
  @Min(100)
  @Max(3000)
  rateBps?: number;

  @IsOptional()
  @IsDateString()
  effectiveTo?: string;
}
```
